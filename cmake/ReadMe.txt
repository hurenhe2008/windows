CMake工程编译助手：

第一步：
    创建源码文件（.h .c .cpp）
    
第二步:
    生成CMakeLists.txt文件（类似linux下的configure）， 编写源码文件关系（遵循CMake语法命令规则）
    
第三步：
    创建编译工程目录(最好与源码文件同级目录)， VS工程可以创建目录为msvc, NMake工程可以创建目录为nmake
    cd msvc
    cd nmake
    
第四步：
    切换当前目录到编译工程目录下，执行命令 
    cmake .\.. -G"Visual Studio 12 2013" (vs2013工程) 
    cmake .\.. -G"NMake Makefiles" (nmake工程：必须用vs development 开发CMD窗口执行，需要cl.exe/link.exe工具)  
    
第五步:
    双击xxx.sln 打开vs IDE集成环境编译生成可执行文件
    nmake /f Makefile 编译生成可执行文件
